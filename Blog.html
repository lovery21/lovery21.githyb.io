¿Como citar un articulo web?

Concepto

Las fuentes bibliográficas son recursos fundamentales para investigaciones y estudios académicos. Incluyen libros, artículos, tesis, revistas, documentos digitales y otros materiales impresos o electrónicos que proporcionan información relevante y verificable. Su cita y referencia adecuadas garantizan la integridad y credibilidad de trabajos científicos y académicos, evitando plagios.

¿Qué es el Plagio y Auto plagio?

Plagio

El plagio es el acto de presentar las palabras, ideas o imágenes de otro como si fueran propias, niega a los autores o creadores de contenido el crédito que se les debe".

Auto Plagio

El auto plagio es la presentación de su propio trabajo previamente publicado como original; como el plagio, el auto plagio no es ético

Requisitos para citar correctamente un articulo web

Para citar un sitio de Internet o un Artículo web existe una lista de referencias bajo los requisitos del estilo APA (7a ed.). Es necesario indicar los siguientes datos en la referencia bibliográfica: datos del autor, fecha de publicación, título del material, título del sitio web y dirección URL (enlace).

Ten en cuenta que el autor también puede ser una organización; por ejemplo: (Asociación Americana de Psicología, 2019). Para concluir el modo de citado para un articulo de Internet debe ser de la siguiente manera:

      Autor(es) del artículo
      Año y mes de publicación
      Título del artículo
      ítulo de la revista
      Número de volumen
      Número de emisión
      Páginas
      URL

Existe la posibilidad de que no se cuenten con todos los requisitos necesarios, para ello se deben aplicar otras condiciones para citar de manera correcta
